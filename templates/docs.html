How It Works

Our Misinformation Detector discriminates between real and fake news using a machine learning pipeline trained on a dataset of over 40,000 articles.

The Dataset

The model was trained on the ISOT Fake News Dataset, which contains both reliable news articles obtained from Reuters and unreliable fabricated news. This diverse collection allows the model to learn linguistic patterns associated with credible reporting versus deceptive writing.

Preprocessing

Before analysis, the input text undergoes a rigorous preprocessing pipeline to prepare it for the machine learning model. First, we perform tokenization and cleaning to break down the text into linguistic units. We then remove "stopwords"â€”common words like "the", "is", and "at" which carry little semantic meaning but appear frequently. Finally, we convert the text into numerical vectors using TF-IDF (Term Frequency-Inverse Document Frequency). This technique weighs terms based on their importance, normalizing against how often they appear across the entire dataset to highlight unique, rigorous keywords.

The Algorithm

We use a Passive Aggressive Classifier, an online learning algorithm that is particularly effective for large-scale text classification tasks. The algorithm derives its name from its update strategy: it remains "passive" when it correctly classifies an input, making no changes to its internal model. However, it turns "aggressive" when a misclassification occurs, significantly adjusting its weight vector to correct the error. This approach allows the model to learn continuously and robustly, adapting quickly to new patterns of misinformation.

Accuracy

During extensive testing and validation, this model demonstrated robust performance, achieving an accuracy of approximately 99% on the held-out test set. This high level of precision makes it a reliable tool for initial credibility screening of news content.
