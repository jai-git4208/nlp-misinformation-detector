API Reference

Integrate our misinformation detection into your own applications using our simple REST API.

Endpoints

POST /analyze

Analyzes the credibility of the provided text.

Request Body

{
  "text": "Your article text or headline here..."
}

Response

{
  "credibility_score": 95.5,
  "assessment": "Low Risk (High Credibility)"
}

Example Usage (cURL)

curl -X POST \
  -H "Content-Type: application/json" \
  -d '{"text": "Breaking news: Water is wet."}' \
  http://127.0.0.1:5000/analyze

GET /status

Checks the health of the API service.

curl http://127.0.0.1:5000/status
